Terrene architecture studio website - dark cinematic design with warm tones

## Design System
- Background: #0b0b0b, Card: #151515, Primary text: #eae6df, Secondary: #b8b3a9, Accent: #e5dfd6
- Display font: Sora (headings, brand), Body font: Space Grotesk
- All colors as HSL in index.css, semantic tokens throughout
- Glass cards: bg-white/5 backdrop-blur-xl border-white/10
- Buttons: pill-shaped (btn-primary, btn-secondary classes)
- No solid borders - depth via shadows and value shifts
- Rounded corners (--radius: 1.5rem)
- Metallic gradient text class: .metallic-text (for footer brand)

## Components
Navbar, Hero, Stats, About, HowWeWork, ProjectFeature, Testimonials, Gallery, BrandSection, Footer, FullscreenMenu, FloatingMenu, SmoothScroll (Lenis)

## Footer Style
- Circular social icons (LinkedIn, Instagram, Dribbble, YouTube)
- Giant metallic gradient "Terrene" text (mixed case)
- Floating center Menu button
- "Developed by — Razal" / cookies / rights reserved bottom bar

## Dependencies
